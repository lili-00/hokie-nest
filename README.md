# College Housing Platform

<!-- Brief introduction to the project -->
Welcome to the College Housing Platform! This project is a web application designed to help students find housing near their college campuses. It is built using React, TypeScript, and Supabase.

## Prerequisites

<!-- List of tools and technologies required to run the project -->
Before you begin, ensure you have met the following requirements:
- **Node.js**: You need Node.js installed on your machine. You can download it from [nodejs.org](https://nodejs.org/).
- **npm**: Node.js comes with npm, which is the package manager used in this project.
- **Supabase CLI**: You need the Supabase CLI to set up a local Supabase instance. Install it from [supabase.io/docs/guides/cli](https://supabase.io/docs/guides/cli).

## Installation

<!-- Step-by-step guide to set up the project -->
1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/college-housing-platform.git
   ```
   Navigate into the project directory:
   ```bash
   cd college-housing-platform
   ```

2. **Install dependencies**:
   Run the following command to install all necessary packages:
   ```bash
   npm install
   ```

3. **Set up Supabase locally**:
   - Initialize Supabase in your project directory:
     ```bash
     supabase init
     ```
   - Start the Supabase local development environment:
     ```bash
     supabase start
     ```

4. **Set up environment variables**:
   - Create a `.env` file in the root directory.
   - Add your Supabase credentials to the `.env` file. These can be found in the `.env` file generated by the Supabase CLI in the `supabase` directory.
   ```env
   VITE_SUPABASE_URL=http://localhost:54321
   VITE_SUPABASE_ANON_KEY=your-anon-key
   ```

   - **Note**: The `VITE_SUPABASE_ANON_KEY` and `VITE_SUPABASE_URL` are automatically generated when you start Supabase locally. You can find them in the `.env` file located in the `supabase` directory.

## Running the Application

<!-- Instructions to run the application -->
1. **Start the development server**:
   Use the following command to start the application:
   ```bash
   npm run dev
   ```
   This will start the Vite development server. You can view the application in your browser at `http://localhost:3000`.

2. **Build for production**:
   To create a production build, run:
   ```bash
   npm run build
   ```

3. **Preview the production build**:
   After building, you can preview the production build using:
   ```bash
   npm run preview
   ```

## Project Structure

<!-- Overview of the project's structure -->
- **src/**: Contains the source code for the application.
  - **components/**: Reusable React components.
  - **pages/**: Different pages of the application.
  - **lib/**: Utility functions and configurations, including Supabase setup.
  - **types/**: TypeScript type definitions.
- **public/**: Static files and assets.
- **supabase/**: SQL migrations and Supabase configuration.
